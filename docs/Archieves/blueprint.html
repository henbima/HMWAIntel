AI-Assisted Build

Cara Paling Efektif: AI sebagai Pair Programmer

Kamu bilang "yang paling sulit adalah memberi AI konteks yang tepat." Untuk project ini, konteksnya sangat terbatas dan clear ‚Äî itu keuntungannya.

üìã Context yang Perlu Kamu Berikan ke AI

Setiap kali mulai session coding baru:

  • 1. Schema Supabase (SQL tables)
  • 2. Module mana yang sedang dikerjakan
  • 3. File/code yang sudah ada (copy-paste existing code)
  • 4. Error message kalau ada bug

Konteksnya sangat bounded ‚Äî tidak seperti HMCS yang massive. Ini project kecil dengan scope jelas.

üéØ Prompt Strategy per Module

  • M1: "Build a Baileys WhatsApp listener that saves all group messages to Supabase. Here's my schema: [paste]. Use @whiskeysockets/baileys v7."
  • M2: "Create Supabase SQL migration for a WhatsApp message tracker. Tables: messages, classified_items, tasks, directions."
  • M3: "Write a Supabase Edge Function that classifies WA messages using Claude API. Categories: task, direction, report, noise. Extract: assignee, deadline, topic."
  • M4: "Create a daily summary cron that queries yesterday's classified items from Supabase and sends a formatted summary via Baileys."
  • M5: "Build a Next.js dashboard with Supabase realtime. Pages: kanban task board, direction search, group activity."

Verdict

Jadi, Build Sendiri atau Tidak?

Build sendiri = jawaban yang tepat untuk kamu. Karena:

‚úì Kamu sudah coding dengan AI ‚Äî progress cepat

‚úì Customisasi = kekuatan utamamu

‚úì Project ini KECIL ‚Äî 5 modul, ~2,000 LOC total

‚úì Banyak repo bisa di-fork ‚Äî bukan dari nol

‚úì Data 100% milik HollyMart

‚úì Bisa jadi fondasi / modul HMCS nanti

‚úì Running cost near-zero

‚úì Konteks untuk AI sangat bounded & clear

Langkah pertama yang saya sarankan:
1. Fork nizarfadlan/baileys-api atau jlucaso1/whatsapp-mcp-ts
2. Setup Supabase free project + buat schema
3. Connect nomor cadangan ‚Üí mulai listen & store semua grup
4. Dalam 2-3 hari, kamu sudah punya Fase 1 running ‚Äî semua chat tersimpan di database
5. Lalu... kamu punya semua waktu di dunia untuk build Fase 2 & 3 secara gradual Jangan tunggu perfect. Mulai dari "listen & store." Sisanya menyusul.
